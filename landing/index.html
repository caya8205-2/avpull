avpull

Download audio/video from YouTube, convert directly to your chosen format — mp3, wav, mp4, and more. No sketchy third-party sites.

Install Read docs
avpull "https://youtu.be/dQw4w9WgXcQ" -f mp3 -q 320

features

△ single command

One command to download and convert. No third-party converter sites, no external binaries.

♪ audio formats

mp3, wav, flac, opus, m4a, aac, ogg — pick what you need.

▣ video formats

mp4, webm, mkv, avi, mov — with quality selection up to 1080p.

⚡ batch mode

Download multiple URLs from a text file, one per line.

⚙ custom quality

Set audio bitrate (192, 320 kbps) or video resolution (720p, 1080p).

💾 custom output

Name your files and choose output directory with -n and -o.

install

One-line install (run in PowerShell as Administrator):

powershell -ExecutionPolicy Bypass -c "irm https://raw.githubusercontent.com/caya8205-2/avpull/main/install.ps1 | iex"

Or download from GitHub Releases.

Install globally via npm:

npm install -g avpull

Clone the repository and build with Bun:

git clone https://github.com/caya8205-2/avpull.git
cd avpull
bun install
bun run build

usage

Basic download

avpull "https://youtu.be/VIDEO_ID"

Downloads and converts to mp3 by default (192 kbps).

Custom format and quality

avpull "https://youtu.be/VIDEO_ID" -f mp4 -q 1080

Download video in mp4 format at 1080p resolution.

Batch download

avpull -b urls.txt
# urls.txt
https://youtu.be/VIDEO_1
https://youtu.be/VIDEO_2
# lines starting with # are ignored

Process multiple URLs from a text file.

Custom output and filename

avpull "https://youtu.be/VIDEO_ID" -o ./downloads -n "my-song" -f wav -q 320

Save to a specific directory with a custom filename.

All options

avpull --help

Display all available flags and options.
